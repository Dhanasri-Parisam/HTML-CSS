This is a simple HTML document.

  • HTML stands for HyperText Markup Language.
  • It is used to create webpages.
  • Basic structure includes: <!DOCTYPE>, <html>, <head>, <body>.

HTML Elements

HTML elements are the building blocks of HTML pages. They consist of tags, which mark the start and end of an element.

For example, a paragraph is defined with the <p> tag: 

<p>This is a paragraph.</p>

Other common elements include:

  • <h1> to <h6> for headings
  • <a> for links
  • <img> for images
  • <div> for sections
