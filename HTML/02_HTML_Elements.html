What are HTML Elements?

An HTML element consists of a start tag, content, and an end tag. Elements can also have attributes that provide extra information.

Example of a paragraph element: <p>This is a paragraph.</p>

  • Block-level elements: <div>, <p>, <h1>
  • Inline elements: <span>, <a>, <img>
  • Empty elements: <img>, <br>
