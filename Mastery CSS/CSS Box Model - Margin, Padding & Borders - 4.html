CSS Box Model

The CSS Box Model is a layout structure that includes the following areas: content, padding, border, and margin.

1. Content

The actual content of the element (text, image, etc.). Its dimensions are set using:

width: 100px;
height: 100px;

2. Padding

Padding creates space inside the element, between the content and the border.

padding: 10px; or padding: 10px 20px 5px 15px;

3. Border

The border wraps around the padding and content. You can define it with:

border-width: 2px;
border-style: solid | dashed | dotted;
border-color: red;
border: 2px dashed magenta; (shorthand)

4. Border Radius

Used to round the corners of a box.

border-radius: 10px; or border-radius: 50%; for a circle

5. Margin

Margin creates space outside the border, separating it from other elements.

margin: 10px; or margin: 5px 10px 15px 20px;

Advanced Box Model Properties

1. box-sizing

box-sizing controls whether padding/border are included in the width and height.

content-box (default): width = content only
border-box: width = content + padding + border

box-sizing: content-box
box-sizing: border-box

2. outline

outline adds a line around an element like a border, but it does NOT affect layout or size.

This has an orange outline

3. overflow

overflow controls what happens when content is too big for the box.

This is a very long text inside a small box that will be hidden when it overflows.

4. max-width, min-width, min-height

These properties control the resizing behavior of boxes.

Resize the window to see max-width and min-height in effect.

10. Text Align

Aligns inline content like text inside a box.

text-align: center;

11. Display

Defines how the box behaves in layout: block, inline, flex, etc.

display: block;
