Color Demo

This page demonstrates different color formats in CSS.

1. Named Color

CSS supports predefined color names like red, teal, blue, etc.

Named Color

2. Hex Color

Hexadecimal values use #RRGGBB format. Each pair represents Red, Green, and Blue.

Hex Color

3. RGB Color

Uses rgb(R, G, B) format where R, G, and B are values from 0 to 255.

RGB Color

4. RGBA Color

Same as RGB but includes an Alpha (opacity) value: rgba(R, G, B, A) where A is 0 to 1.

RGBA Color

5. HSL Color

Stands for Hue, Saturation, Lightness: hsl(H, S%, L%).

HSL Color

6. HSLA Color

Like HSL but includes Alpha (opacity): hsla(H, S%, L%, A).

HSLA Color
