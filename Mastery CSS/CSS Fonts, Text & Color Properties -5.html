CSS Fonts, Text & Color Properties

This page demonstrates common styling used to control fonts, text appearance, and color in CSS.

1. font-family

font-family defines the typeface used for text.

This text uses a custom font family.

2. font-size

font-size sets the size of the text. Values can be in px, em, rem, or %.

This text has a larger font size.

3. font-weight

font-weight controls the thickness of the text.

This text is bold.

4. font-style

font-style allows text to be italic or normal.

This text is italic.

5. text-align

text-align sets horizontal alignment: left, center, right, or justify.

This text is centered.

6. text-decoration

text-decoration adds effects like underline, overline, or line-through.

This text is underlined.

7. text-transform

text-transform changes the case: uppercase, lowercase, or capitalize.

this text is capitalized.

letter-spacing controls the space between letters.
