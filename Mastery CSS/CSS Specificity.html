CSS Specificity & Cascading

Specificity determines which CSS rule takes precedence when multiple rules target the same element.

Cascading means the final style is determined by the source order and specificity level.

Specificity Hierarchy:

  • Inline styles (e.g., style="") → Most specific
  • #id selectors
  • .class, :pseudo-class, [attributes]
  • Element and pseudo-element selectors → Least specific
