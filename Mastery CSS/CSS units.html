CSS Units

CSS supports both absolute and relative units for sizing elements, fonts, margins, padding, etc.

1. Absolute Units

These units are fixed and not affected by screen size:

  • px – Pixels (e.g., 16px)
  • in – Inches (1in = 96px)
  • cm – Centimeters (1in = 2.54cm)
  • mm – Millimeters
  • pt – Points (1pt = 1/72 inch)
  • pc – Picas (1pc = 12pt)

2. Relative Units

These scale based on parent or screen size:

  • % – Percentage of parent element
  • em – Relative to parent font size
  • rem – Relative to root (html) font size
  • vw – Relative to 1% of viewport width
  • vh – Relative to 1% of viewport height
  • vmin – Relative to smaller of viewport width or height
  • vmax – Relative to larger of viewport width or height
  • ex – Relative to x-height of current font
  • ch – Relative to width of “0” (zero) character
